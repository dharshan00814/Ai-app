# 🚀 FIX CONNECTION ERROR - COMPLETE GUIDE

## What You're Seeing
```
🔴 Server Connection Error
Cannot reach server at http://localhost:3000
The backend server is not running.
```

---

## ✅ WHY THIS HAPPENS

The website (frontend) is running, but the **backend server is NOT running**.

**Server won't run because:** Node.js is not installed

---

## 🎯 FIX IN 3 MINUTES

### **1️⃣ Install Node.js (If not already installed)**

**For Windows:**

a) Open browser: **https://nodejs.org/**

b) Click **LTS** button (recommend v20.x)

c) Click **Download for Windows**

d) Run the downloaded `.msi` file

e) In installer:
   - Click **Next**
   - Accept License ✅
   - Keep default path
   - ✅ **CHECK:** "Automatically install necessary tools"
   - Click **Install**
   - Wait for completion
   
f) **RESTART YOUR COMPUTER** (very important!)

---

### **2️⃣ Verify Node is Installed**

After restart, open **PowerShell** and type:

```powershell
node --version
npm --version
```

**You should see version numbers like:**
```
v20.10.0
9.8.1
```

If you see errors, close and open a **NEW PowerShell** window.

---

### **3️⃣ Start the Backend Server**

In PowerShell:

```powershell
cd "c:\Users\Dharshan\OneDrive\Desktop\ai app"
npm install
npm start
```

**WAIT for this message:**
```
============================================================
🤖 AI Resume Screening System Started
============================================================
✓ Server running at: http://localhost:3000
✓ API Base URL: http://localhost:3000/api
```

When you see this ✅ **Your server is running!**

---

### **4️⃣ Refresh Your Browser**

- Keep PowerShell window **OPEN** (server must keep running)
- Go back to browser
- Press: `Ctrl + F5` (or `Cmd + Shift + R` on Mac)
- Click **Retry Connection** button

**You should now see the resume upload interface!** ✅

## ❌ Troubleshooting

### Problem: "npm: not recognized" even after installing Node.js

**Solution:**
1. Close PowerShell completely
2. Restart your computer
3. Open a NEW PowerShell
4. Try `npm --version` again

### Problem: "Port 3000 already in use"

**Solution:**
```powershell
# Find process using port 3000
netstat -ano | findstr :3000

# Kill that process (replace 1234 with actual PID)
taskkill /PID 1234 /F

# Try npm start again
```

**Alternative:** Use different port:
```powershell
# Edit .env file and change: PORT=3001
# Then run: npm start
```

### Problem: Still getting connection error after all steps

**Solutions:**
1. Make sure PowerShell with `npm start` is still open
2. Try refreshing browser (`Ctrl + F5`)
3. Check browser console (`F12`) for specific errors
4. If you see "Cannot find module 'express'":
   ```powershell
   npm install
   npm start
   ```

### Problem: "Module not found" errors

**Solution:**
```powershell
# Delete old packages
Remove-Item node_modules -Recurse -Force
Remove-Item package-lock.json

# Reinstall fresh
npm install
npm start
```
