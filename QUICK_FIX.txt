# QUICK FIX - Upload & Connection Errors

## 🚨 Main Issue
**Node.js is not installed!** This is why you can't upload or connect.

## ⚡ Quick Fix (5 minutes)

### 1. Install Node.js
- Go to: https://nodejs.org/
- Download **LTS version**
- Run installer with default settings
- **RESTART your computer**

### 2. Verify Installation
```powershell
node --version
npm --version
```

### 3. Install Dependencies
```powershell
cd "c:\Users\Dharshan\OneDrive\Desktop\ai app"
npm install
```

### 4. Start Server
```powershell
npm start
```

### 5. Open Browser
```
http://localhost:3000
```

## 🔴 If Still Not Working

### Error: `npm: not recognized`
→ **Node.js not installed** - Follow Step 1 & 2 above

### Error: `Cannot connect to localhost:3000`
→ **Server not running** - Check PowerShell window, run `npm start`

### Error: `Failed to fetch`
→ **Server crashed** - Run `npm start` again, check for errors

### Error: `Invalid file type`
→ **Wrong format** - Use PDF, DOCX, DOC, or TXT files only

### Error: `API key error`
→ **Check .env file** - Verify OPENAI_API_KEY is correct
