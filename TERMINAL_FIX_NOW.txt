## 🔧 Common PowerShell Issues & Quick Fixes

### Issue: "scripts disabled on this system"
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
# Press: Y and Enter
```

### Issue: "still not recognized after restart"
```powershell
# Close ALL PowerShell windows
# Restart your computer (full restart)
# Open NEW PowerShell window
# Try: node --version
```

### Issue: "Port 3000 already in use"
```powershell
# Option 1: Stop current process
netstat -ano | findstr :3000
# Note the PID
taskkill /PID [PID_HERE] /F

# Option 2: Use different port
# Edit .env and change: PORT=3001
# Then: npm start
```

### Issue: "npm cache error"
```powershell
npm cache clean --force
npm install
```
