🚀 QUICK START (3 STEPS)

Step 2: Start the Server
  npm start

Step 3: Open in Browser
  http://localhost:3000


📊 HOW IT WORKS
===============

1. UPLOAD RESUMES
   ↓
2. AI ANALYSIS (Using OpenAI GPT)
   ├─ Approves if score ≥ 60%
   └─ Rejects if score < 60%
   ↓
3. APPROVED → SEND TO PRINCIPAL
   ↓
4. PRINCIPAL REVIEWS & MAKES FINAL DECISION

🎯 SCORING GUIDE
================

90-100%  🟢 EXCELLENT  → Highly recommended for principal review
70-89%   🟡 GOOD       → Strong candidate
50-69%   🟠 FAIR       → Some concerns noted
Below 50% 🔴 POOR       → Does not meet criteria


🛠️ TROUBLESHOOTING
===================

Q: Port 3000 already in use?
A: Change PORT in .env file to 3001 or 3002, then restart

Q: "Module not found" error?
A: Run: npm install

Q: API errors?
A: Check .env for valid OpenAI API key and internet connection

Q: Resume upload fails?
A: Check file size (max 10MB) and format (PDF, DOCX, DOC, TXT)
